Pantry Perfect

React / Front-End Capstone

Pantry Perfect

View Site! View the Code

An application that allows a user to track the quantity of any groceries they choose, create recipes they commonly cook, link the ingredients to their tracked items, and build a grocery list on the fly.

Built in React.js and utilizing a heavily edited Semantic-UI for React style library throughout, the application also makes use of Charts.js for React and Tilt.js for React for statistics. User data is populated from a JSON file hosted off of a JSON Server. Both the application and the server are hosted on Heroku.

Use By

React / Nash Hack Weekathon Winner

Use By

View Site! View the Code

Use By is a collaborative project that was born out of the Nash Hack Weekathon! Originally pitched by Angie Renfro, I was given the opportunity to be the team lead for the project with Angie at the helm of UI/UX control. I lead the team through the 48 hour build window to deliver a MVP prototype for an application that allows a user to track the expiration dates of common foods and see their money waste when food gets thrown out.

Use By is built in React.js and utilizes basic CSS for media query and Semantic UI for React. A simple JSON file with a flat datastructure is hosted off of a JSON server. Both Use By and its server are deployed to Heroku.

NutShell

React / Front-End Group Project

NutShell

View Site! View the Code

Nutshell is a collaborative week long group project from Nashville Software School to build a prototype social media web application that allows a user to create, edit, and delete articles, events, and tasks, as well as feature a intra-network chat room style messaging system and the ability to add other users as friends and see their page!

Nutshell was built in React.js and utilized two style libraries, Bloomer and ReactStrap (also extensive CSS) to visualize the code. The goal was to engage an agile approach to coordinate tasks amongst a 4 person team and utilize a flat JSON datastructure for demonstration purposes. My responsibilities included the Login, User Profile, Events, and Friends section of the application.

Bangazon

C#/.NET / Back-End Group Project

Bangazon!

View the Code

Bangazon is a collaborative week long group project from Nashville Software School to build a prototype e-commerce web application that allows a user to create product listings, utilize search features for them, and actually complete a purchase experience via a shopping cart!

Bangazon was built in C#/ASP.NET MVC WebApp and utilized the ORM Entity Framework primarily, with Identity for authentication and authorization of Users. A SQL Server database is used for the project. The goal was to follow SCRUM methodology to coordinate tickets amongst a 4 person team to accomplish our Sprint projection. My responsibilities included the Initializing the database with starter data, building the Search Bar, building the Product Category index view, Shopping Cart item removal, and total order cancellation.
