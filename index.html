Pantry Perfect

Category / React

Pantry Perfect

Visit Now! View the Code

An application that allows a user to track the quantity of any groceries they choose, create recipes they commonly cook, link the ingredients to their tracked items, and build a grocery list on the fly.

Built in React.js and utilizing a heavily edited Semantic-UI for React style library throughout, the application also makes use of Charts.js for React and Tilt.js for React for statistics. User data is populated from a JSON file hosted off of a JSON Server. Both the application and the server are hosted on Heroku.

Use By

Category / React

Use By

Visit Now! View the Code

Use By is a collaborative project that was born out of the Nash Hack Weekathon! Originally pitched by Angie Renfro, I was given the opportunity to be the team lead for the project with Angie at the helm of UI/UX control. I lead the team through the 48 hour build window to deliver a MVP prototype for an application that allows a user to track the expiration dates of common foods and see their money waste when food gets thrown out.

Use By is built in React.js and utilizes basic CSS for media query and Semantic UI for React. A simple JSON file with a flat datastructure is hosted off of a JSON server. Both Use By and its server are deployed to Heroku.
