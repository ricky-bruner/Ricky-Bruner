I'm a former retail manager, car builder, apprentice electrician, and customer service wizard who acknowledged a love for creating and learning and realized that Software Development was the home long sought after. Development gives me the opportunity to pour my creative energy into something tangible, with countless options for any one idea, and endless learning potential. I love dreaming up ideas and realizing them in the front-end, but also the mental challenge of creating a strong back-end to support the whole dream. Preference-less, I genuinely enjoy all sides of coding, from the immense feelings of accomplishment to the crushing failures that we learn so much from, that all go hand in hand with development.

Throughout my career, I've taken every challenge as potential to improve myself. I've ascended to leadership roles numerous times because I love communicating and working with teams to tackle challenges and not only training and teaching others, but learning so much from them as well. The value of working hard is perhaps the most essential lesson I've gleaned through my various positions. I believe that it takes a wealth of varying experiences and great open-mindedness to grow internally. With that, I welcome any and all advice! Please reach out to me for anything through whichever means you prefer!

Throughout my career, I've always been driven by hard work and gaining new skills. I love positions that challenge my ability and allow me to grow and expand my skillset in new and exciting ways! First suggested to me by my brother-in-law and fellow dev Terrence, I began looking into the tech industry and researching what this crazy new world (to me!) was all about. I soon found myself in love with the challenge of understanding the complexities of coding, and driven by my own creativity to build anything I could imagine.

I've always felt a strong pull towards organization and I love that code allows me to test these principles in new and challenging ways. I love thinking up and creating ways for clients to interact with technology, so front-end development checks a lot of boxes for me. Back-end development affords me the challenge of organizing data and critically thinking my way through all sorts of unique problems. In all, developement offers me so much that I never believed I'd find in an industry and job, and I'm so excited at the opportunity to work in it! I cherish advice and love sharing stories, please feel free to contact me in whatever form works best for you!

Pantry Perfect

Category / React

Pantry Perfect

Visit Now! View the Code

An application that allows a user to track the quantity of any groceries they choose, create recipes they commonly cook, link the ingredients to their tracked items, and build a grocery list on the fly.

Built in React.js and utilizing a heavily edited Semantic-UI for React style library throughout, the application also makes use of Charts.js for React and Tilt.js for React for statistics. User data is populated from a JSON file hosted off of a JSON Server. Both the application and the server are hosted on Heroku.

Use By

Category / React

Use By

Visit Now! View the Code

Use By is a collaborative project that was born out of the Nash Hack Weekathon! Originally pitched by Angie Renfro, I was given the opportunity to be the team lead for the project with Angie at the helm of UI/UX control. I lead the team through the 48 hour build window to deliver a MVP prototype for an application that allows a user to track the expiration dates of common foods and see their money waste when food gets thrown out.

Use By is built in React.js and utilizes basic CSS for media query and Semantic UI for React. A simple JSON file with a flat datastructure is hosted off of a JSON server. Both Use By and its server are deployed to Heroku.

NutShell

Category / React

NutShell

Visit Now! View the Code

Nutshell is a collaborative week long group project from Nashville Software School to build a prototype social media web application that allows a user to create, edit, and delete articles, events, and tasks, as well as feature a intra-network chat room style messaging system and the ability to add other users as friends and see their page!

Nutshell was built in React.js and utilized two style libraries, Bloomer and ReactStrap (also extensive CSS) to visualize the code. The goal was to engage an agile approach to coordinate tasks amongst a 4 person team and utilize a flat JSON datastructure for demonstration purposes. My responsibilities included the Login, User Profile, Events, and Friends section of the application.
